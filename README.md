# Aplikacja_projekt

React Native + Expo training log app scaffold that uses Supabase authentication. Includes:

- AuthContext wrapping Supabase email/password auth
- Native stack for Login/Register
- Bottom tabs for Home, Add, Stats, Profile with Ionicons

## Getting started

```bash
npm install
npm start
```

Remember to configure `EXPO_PUBLIC_SUPABASE_URL` and `EXPO_PUBLIC_SUPABASE_ANON_KEY` in your environment (e.g., `.env`) before running the app.
